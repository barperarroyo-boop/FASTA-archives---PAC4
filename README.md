Estos archivos se pueden leer con .txt o bloc de notas. Son dos archivos de secuencias de nucleótidos generados aleatoriamente
mediante R. El formato FASTA es el formato de texto plano generalmente utilizado para representar secuencias de nucleótidos o aminoácidos
y tiene como terminación .fasta. Siempre tiene un encabezado > seguido de la secuencia
